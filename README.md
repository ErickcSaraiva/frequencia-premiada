# frequencia-premiada
O projeto Frequência Premiada é um MVP de plataforma que utiliza tags NFC para registrar a entrada e saída de alunos em escolas públicas, com o objetivo de reduzir a evasão escolar. A solução combina registro confiável de frequência com gamificação (pontos por presença, bônus por semana completa e streak – dias consecutivos). No MVP, os alunos são cadastrados pela escola e recebem uma tag NFC; um adulto (porteiro ou professor) realiza a leitura em um aplicativo Android. A direção acompanha a frequência, os pontos e os alertas de 3 faltas consecutivas (indicativo de evasão) em um painel web. O projeto é de baixo custo, focado em engajamento da comunidade escolar e geração de dados para ação preventiva.
